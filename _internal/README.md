# _internal/ — Archivos privados

Esta carpeta contiene material **interno** que NO debe formar parte del sitio público.

- `Backup/` — Versiones antiguas/respaldos del sitio.
- `SEO y marketing/` — PDFs de estrategia, notas y documentación.

**Bloqueado en `robots.txt`** (`Disallow: /_internal/`) para que ningún buscador indexe esta ruta.
GitHub Pages igual sirve los archivos aquí en `https://alveos.mx/_internal/...`, así que **no pongas datos sensibles** (contraseñas, datos de pacientes, etc.) en esta carpeta. Para eso, mueve los archivos completamente fuera del repo.
